1. **JP оригинал** = `D:\Steam\...\Persistent\master\master.mdb` (SQLite, таблица

3. **RU** = `localized_data/text_data_dict.json` + глоссарий `_ПРОГРЕСС.md`. КАНОН.

## ⚙️ РАБОЧИЙ ПРОЦЕСС применения
Пиши скрипт `_apply_*.py` (gitignored): dict `{JP-ключ или idx: RU}` → дозапись в нужную
категорию `d.setdefault("CAT",{})[idx]=ru`, нормализация **ё→е** (`yo()`), проверка отсутствия
японского в результате (`JP=re.compile(r"[ぁ-ゖァ-ヺ一-鿿]")`), баланс тегов `<...>`,
финальный `json.load` для валидации. Для навыков/гонок ключуй по JP-строке (ловит дубликаты-
редкости на разных индексах). Всегда сначала DRY-прогон с образцами, потом запись.

#### 📒 КАНОН ОПИСАНИЙ cat 48 (соблюдать строго — выяснено сверкой с существующими)
- **Интенсивность:** わずかに/ちょっと/ほんの少し → **«Слегка»**; 少し/少しずつ → **«Умеренно»**;
  すごく/ものすごく/かなり/大きく/とても → **«Значительно»** (или «колоссально» для ものすごく в крафте);
  без слова → базовый глагол с заглавной («Повышает»).
- **Эффекты:** 速度が上がる→«повышает скорость», 加速力が上がる→«повышает ускорение»,
  持久力を回復→«восстанавливает выносливость», 疲れにくくなる→«снижает усталость»,
  前に出る→«продвигается вперёд», 速度を上げ続ける→«повышает скорость продолжительно»,
  スピードとパワーが上がる→«повышает Скорость и Силу», コース取りがうまくなる→«лучше проходит дистанцию»,
  萎縮させる→«сковывает соперников».
- **Условия:** レース序盤/中盤/終盤に→«в начале/середине/конце гонки», 終盤が迫ったとき→
  «при приближении к концу гонки», 終盤以降→«в конце гонки и позже», 後半→«во второй половине гонки»,
  スタート時→«на старте», 直線で→«на прямой», コーナーで→«на повороте», 最終直線で→«на финишной прямой»,
  最終コーナーで→«на финальном повороте», 下り坂/上り坂→«на спуске/подъёме»,
  前方/前の方にいると→«находясь впереди», 後方→«находясь сзади», 中団→«в середине группы»,
  中団以降→«в середине группы или дальше», 好位置→«при хорошей позиции», 先頭→«в лидерах»,
  内ラチ側→«у внутреннего ограждения», ラストスパートの最中→«во время финального рывка»,
  追い抜くと→«при обгоне», 競り合うと→«в борьбе с соперниками», 余力十分なら→«имея запас сил»,
  残りNm→«на последних N м / на отметке N м», Nバ身以内→«в пределах N корпусов».
- **Длительность** しばらくの間/少しの間/短い間/超長時間 — в движке УБИРАЕТСЯ (для чистоты);
  в крафте можно «ненадолго».
- **ТЕГИ — КОРОТКАЯ форма** (важно, отличается от факторов!): ＜短距離＞→«(Спринт)»,
  ＜マイル＞→«(Миля)», ＜中距離＞→«(Средняя)», ＜長距離＞→«(Длинная)», ＜ダート＞→«(Грунт)»,
  ＜作戦・逃げ/先行/差し/追込＞→«(Лидер/Преследователь/Догоняющий/Замыкающий)». Комбо через «/».
- **⚠️ ВАЖНО:** существующие переводы золотых навыков КОНТЕКСТНО-художественны (одно «速度が上がる»
  переводится по-разному: «способность вырваться из группы», «увеличивает отрыв» и т.п.). Это
  НЕвоспроизводимо механически — поэтому золотые ТОЛЬКО руками. Движок — только для формульных.
- **Структура движка:** «{Интенс}{глагол} {место без запятой}, {позиция} {время}. (тег)».
  Пример: «Слегка повышает скорость, находясь впереди при приближении к концу гонки. (Грунт)».
